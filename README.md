# Startup Idea Validator Bot

## Overview

Startup Idea Validator Bot is a web application that evaluates startup ideas and provides business insights including:

* Startup Health Score
* SWOT Analysis
* Success Probability
* Risk Assessment
* Investor Verdict
* Startup DNA Analysis

## Tech Stack

* React.js
* Flask
* Python
* Axios
* REST API

## Features

* Dynamic Startup Scoring
* SWOT Analysis
* Success Probability Meter
* Risk Level Detection
* Investor Verdict
* Interactive Dashboard

## How to Run

### Backend

```bash
python app.py
```

### Frontend

```bash
npm start
```
